Ejercicio 2 – PreferenceScreen (SharedPreference2)

Aplicación Android con:
- Pantalla principal con EditText + botón PREVIEW + TextView de previsualización + botón **CLOSE**
- Pantalla Settings (Navigation Drawer) con PreferenceScreen:
  - Basic Format: tamaño, color de texto (FG), color de fondo (BG), negrita y cursiva
  - Advanced Format: alpha y rotación (si procede)

 Flujo de uso
1. En Main, se muestra un texto inicial (por defecto).
2. Pulsar PREVIEW aplica el texto escrito (o (preview) si está vacío) y aplica el formato configurado en Settings.
3. Pulsar CLOSE limpia el EditText y deja el TextView en (preview).

 Ejecución
Abrir el proyecto en Android Studio y ejecutar en un emulador/dispositivo.
